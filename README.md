# University Student Dashboard

This project analyzes university student data focusing on admission, retention, and satisfaction indicators.

Developed with **Python, Pandas, Matplotlib, Seaborn, and Streamlit** as part of the Data Visualization and Dashboard Deployment activity at Universidad de la Costa.

##  Features
- Retention rate trends over time
- Student satisfaction by year
- Comparison between Spring and Fall terms
- Interactive filters (Year, Term)
